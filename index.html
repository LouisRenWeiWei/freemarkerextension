Freemarker extension

Freemarker Extension is an operator for the Rapidminer software. It can apply a freemarker template on the Rapidminer's exampleset. You can create html, xml, csv etc.. outputs from the rapidminer data with this extension. It can be useful to create scheduled business reports and send them in email, use the process as a servlet with fancy html output, or use an xhtmlrenderer to create pdf documents.

The basic html template (without css and javascript) file main.ftl is the following: 

 
    <!DOCTYPE html>
    <html>
      <head>
        <title>${title}</title>
      </head>
      <body>
        <div id="wrapper">
          <H1>${title}</H1>
          
          <table id="keywords" cellspacing="0" cellpadding="0">
            <thead>
              <tr>
              <#list columns as col>
                <th class="header"><span>${col.name}</span></th>
              </#list>
              </tr>
            </thead>
            <tbody>
            <#list rows as row>
              <tr>
              <#list row.cells as cell>
                <td>${cell}</td>
              </#list>
              </tr>
            </#list>
            </tbody>
          </table>
        </div> 
      </body>
    </html>
